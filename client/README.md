# Oasis Crisis Response Platform

A cutting-edge, professional crisis response ecosystem built for the Google Solution Challenge.

## About the Project 🌍
Oasis is a comprehensive crisis management platform designed to bridge the critical gap between disaster victims (Seekers), ground responders (Volunteers), and overseeing bodies (Authorities). In times of emergency, rapid communication and resource allocation are matters of life and death. 

Our platform tackles these challenges by providing:
- **For Seekers / Victims:** A streamlined, highly accessible portal to quickly request emergency aid, share vital information, and receive live updates.
- **For Volunteers:** A dedicated hub to track ongoing rescue missions, manage logistics, and coordinate effectively with other on-ground personnel.
- **For Authorities (Mission Control):** A "God-level" centralized dashboard of the crisis zone. It aggregates real-time data, maps critical incidents, and helps orchestrate large-scale responses.

Beyond just functionality, Oasis prioritizes an **"Urgent Serenity"** design philosophy. We ensure that the User Interface remains calm, highly readable, and cinematic even in the most chaotic and high-stress emergency conditions.

## Key Capabilities 🛠️
- **Real-Time Mission Control**: Centralized mapping and data dashboard for high-stakes decision-making.
- **AI Field Intelligence**: Smart, natural-language input systems allowing responders to rapidly log details from the field without fighting complex forms.
- **Seamless Volunteer Coordination**: Intelligent onboarding and mission assignment for civilian responders.
- **Glass-Bento Design Architecture**: Modern, premium UI utilizing glassmorphism and multi-layered depth effects to reduce cognitive load.

## Tech Stack 💻
- **Frontend Framework:** React + Vite
- **Styling:** Tailwind CSS (featuring a strict 8px visual rhythm)
- **Animations:** Framer Motion (tactile micro-interactions)
- **Iconography:** Lucide React

## Getting Started 🚀
1. Clone the repository: `git clone https://github.com/CoderShreya16/Oasis.git`
2. Install dependencies: `npm install`
3. Start the application: `npm run dev`
